# Vigneshwaran S – Portfolio

A modern and responsive personal portfolio built with React, TypeScript, Vite, and Tailwind CSS to showcase my projects, skills, achievements, and experience.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion

## Run Locally

```bash
git clone https://github.com/Vikki-2006/Portfolio.git
cd Portfolio
npm install
npm run dev
```
